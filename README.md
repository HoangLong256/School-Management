# School-Management

## Description
This application is designed to manage the the courses as well as staffs of the School of Science and Technology, RMIT.
The details description can be found in the Assignment.pdf file

## Features
- The program should provide a graphical user interface to the functionality 
- Ability to add new courses to the system 
- Ability to add new units to the system 
- Ability to assign units to courses
- Ability to assign staff members to courses and units 
- Ability to delete courses 
- Ability to delete units 
- Ability to see a list of units available for the course 
- Ability to display the details of a course 
- Ability to display the details of a unit
- Ability to remember all information between consecutive runs of the program 
- Ability to initialise a list of staff members with some pre-determined data
